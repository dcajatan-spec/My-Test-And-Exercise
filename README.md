# 📚 My Test And Exercise

Welcome to my collection of beginner-friendly web development projects! This repository contains simple yet practical exercises that showcase fundamental HTML, CSS, and JavaScript concepts.

## 📁 Projects Overview

### 1️⃣ **1st Project** (`1st proj/`)
A simple beginner project demonstrating basic JavaScript functionality.

**Files:**
- `index.html` - Main HTML structure
- `try.js` - JavaScript logic and interactivity

**What you'll learn:**
- Basic DOM manipulation
- JavaScript fundamentals
- HTML-JS integration

**How to use:**
1. Clone or download this repository
2. Navigate to the `1st proj` folder
3. Open `index.html` in your web browser
4. Review the code to understand the implementation

---

### 2️⃣ **2nd Project** (`2nd proj/`)
An interactive project that demonstrates error handling and visual elements.

**Files:**
- `index.html` - Main HTML structure
- `error.js` - JavaScript with error handling logic
- `1.png` - Supporting image asset

**What you'll learn:**
- Error handling in JavaScript
- Working with images and assets
- User interaction patterns

**How to use:**
1. Navigate to the `2nd proj` folder
2. Open `index.html` in your browser
3. Study the `error.js` file to understand error handling
4. Reuse the error handling patterns in your own projects

---

### 3️⃣ **3rd Project** (`3rd proj/`)
A calculator project that demonstrates function creation and mathematical operations in JavaScript.

**Files:**
- `index.html` - Calculator HTML interface
- `calc.js` - Calculator logic and functions

**What you'll learn:**
- Function creation and usage
- Event handling
- Mathematical operations
- Input/output management

**How to use:**
1. Open `3rd proj/index.html` in your browser
2. Test the calculator functionality
3. Review `calc.js` to understand how functions handle calculations
4. Adapt the code for your own calculator projects

---
